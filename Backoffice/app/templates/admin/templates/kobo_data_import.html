Upload an Excel file exported from KoBoToolbox. The wizard will detect the form structure, questions, and data types automatically.

How to export data from KoBo:

  1. Open your form in KoBoToolbox
  2. Go to Data → Downloads
  3. Select XLS format and click Export
  4. Download the generated file and upload it here
