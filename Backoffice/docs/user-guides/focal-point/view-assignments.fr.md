# Voir vos missions

Votre tableau de bord affiche les formulaires qui vous sont assignés (et leur statut).

## Avant de commencer

- Vous avez besoin d'un accès **Point focal** (ou un administrateur agissant comme point focal).
- Vous devez être assigné à au moins un **pays**.

## Étapes

1. Ouvrez votre **Tableau de bord**.
2. Examinez vos cartes de missions (titre, date limite et statut).
3. Cliquez sur une mission pour ouvrir le formulaire.

## Comprendre le statut de la mission

| Statut | Signification |
|--------|---------------|
| **En attente** | Pas encore commencé |
| **En cours** | Commencé mais non soumis |
| **Soumis** | Terminé et soumis |
| **En retard** | Date limite dépassée, non soumis |

## Conseils

- Commencez tôt pour éviter les problèmes de dernière minute.
- Votre progression est généralement **enregistrée automatiquement** pendant que vous travaillez.
- Si vous ne voyez pas une mission attendue, contactez votre administrateur.
