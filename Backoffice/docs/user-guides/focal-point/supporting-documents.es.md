# Documentos de respaldo (punto focal)

Usa esta guía si tu asignación te pide cargar archivos de evidencia o fuente junto con el formulario.

## Antes de comenzar

- Confirma qué documentos son requeridos para tu asignación (si los hay).
- Asegúrate de que tus archivos estén listos (y no protegidos con contraseña a menos que se requiera).

## Cargar documentos

1. Abre tu asignación desde el **Panel de control**.
2. Busca una sección como **Documentos**, **Adjuntos** o **Documentos de respaldo**.
3. Haz clic en **Cargar** (o **Agregar archivo**).
4. Selecciona tu archivo y confirma.
5. Repite para cualquier documento adicional.

## Consejos

- Usa nombres de archivo claros (ejemplo: `Country_Period_Source.pdf`).
- Si un documento es grande, cárgalo primero (para no perder tiempo al final).
- No cargues datos personales sensibles a menos que tu organización lo requiera explícitamente.
- Si te piden cargar una hoja de cálculo, evita pestañas/hojas ocultas adicionales a menos que sea necesario (manténlo simple).

## Problemas comunes

- **No puedo ver el área de carga**: puede que no tengas el rol correcto para cargar documentos—pregunta a tu administrador.
- **La carga falla**: intenta con un archivo más pequeño, verifica tu conexión a Internet o usa un formato de archivo diferente.
- **No estoy seguro de qué cargar**: relee las instrucciones de la asignación y pregunta a tu administrador qué significa "evidencia requerida" para esta asignación.
