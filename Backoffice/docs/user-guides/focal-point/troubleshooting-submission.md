# Troubleshooting submission issues (Focal point)

Use this guide when you can open an assignment but you can’t submit successfully (or your submission seems blocked).

## Before you start

- Confirm you are working on the correct assignment and period.
- If possible, keep a copy of key numbers/answers (so you can re-enter them if needed).

## I clicked “Submit” and nothing happens

Try this:
1. Scroll to the top and bottom once to look for a red/amber validation message.
2. Check each section for missing required fields.
3. Refresh the page once and try again.

If it still fails, contact your admin/support and share:
- assignment name
- what section you were in
- a screenshot (including any message)

## The form says required fields are missing (but I can’t find them)

Common causes:
- a required field is in another section
- a matrix/table has a required cell
- the field is visible only after a previous answer (conditional logic)

What to do:
- Use the section navigation and open each section that shows a warning.
- Look for fields marked with an asterisk (*) or highlighted borders.

## I submitted, but now I can’t edit

This is usually normal.

After submission, editing is often locked until an admin/approver reopens it.

See: [Submission statuses and what you can do](../common/submission-statuses-and-permissions.md)

## My assignment disappeared / I can’t find it anymore

Possible reasons:
- the assignment is no longer active (closed)
- your country access changed
- you were removed from the assignment roles

What to do:
- Check your dashboard and filters (if available)
- Contact your admin and ask them to confirm:
  - your country access
  - your assignment role (data entry)
  - the assignment is still active for your country

## Uploading a supporting document fails

Try this:
- reduce the file size (export PDF as “small size”, compress images)
- rename the file (simple letters/numbers, no special characters)
- try a different browser

If it still fails, share with support:
- file name, type, and size
- the assignment name
