# Soumettre des données de formulaire

Utilisez ce guide pour compléter une mission et soumettre votre formulaire.

## Avant de commencer

- Vous avez besoin d'une **mission actuelle**.
- Ayez vos données prêtes (et sources, si vous devez ajouter des commentaires).

## Étapes

1. Depuis votre **Tableau de bord**, ouvrez la mission que vous voulez compléter.
2. Utilisez la barre latérale gauche pour naviguer entre les **sections** du formulaire.
3. Remplissez les champs. Les champs requis sont marqués d'un astérisque (*).
4. Vérifiez les messages de validation et corrigez-les.
5. Lorsque vous êtes prêt, cliquez sur **Soumettre** pour finaliser vos données.

## Enregistrer votre progression

- Votre travail est généralement enregistré au fur et à mesure.
- Vous pouvez partir et revenir plus tard.
- Si vous voyez une option **Enregistrer brouillon**, vous pouvez l'utiliser pour enregistrer explicitement.

## Conseils

- Utilisez les champs de commentaires pour documenter les sources de données (si disponible).
- Si vous ne pouvez pas soumettre, recherchez un champ requis manquant ou une erreur de validation.
