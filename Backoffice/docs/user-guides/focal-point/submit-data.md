# Submit Form Data

Use this guide to complete an assignment and submit your form.

## Before you start

- You need a **current assignment**.
- Have your data ready (and sources, if you need to add comments).

## Steps

1. From your **Dashboard**, open the assignment you want to complete.
2. Use the left sidebar to navigate between **sections** of the form.
3. Fill in the fields. Required fields are marked with an asterisk (*).
4. Check for any validation messages and fix them.
5. When you’re ready, click **Submit** to finalize your data.

## Saving Your Progress

- Your work is usually saved as you go.
- You can leave and return later.
- If you see a **Save draft** option, you can use it to save explicitly.

## Tips

- Use comment fields to document data sources (if available).
- If you can’t submit, look for a missing required field or a validation error.
