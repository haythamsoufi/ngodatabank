# Enviar datos de formulario

Usa esta guía para completar una asignación y enviar tu formulario.

## Antes de comenzar

- Necesitas una **asignación actual**.
- Ten tus datos listos (y fuentes, si necesitas agregar comentarios).

## Pasos

1. Desde tu **Panel de control**, abre la asignación que quieres completar.
2. Usa la barra lateral izquierda para navegar entre las **secciones** del formulario.
3. Completa los campos. Los campos requeridos están marcados con un asterisco (*).
4. Verifica si hay mensajes de validación y corrígelos.
5. Cuando estés listo, haz clic en **Enviar** para finalizar tus datos.

## Guardar tu progreso

- Tu trabajo generalmente se guarda mientras avanzas.
- Puedes salir y volver más tarde.
- Si ves una opción **Guardar borrador**, puedes usarla para guardar explícitamente.

## Consejos

- Usa campos de comentarios para documentar fuentes de datos (si están disponibles).
- Si no puedes enviar, busca un campo requerido faltante o un error de validación.
