# Ajouter un Nouvel Utilisateur

Utilisez ce guide pour créer un nouveau compte Backoffice pour un administrateur ou un point focal.

## Avant de commencer

- Vous devez avoir un accès **Admin**.
- Pour un **point focal**, vous devez connaître le(s) **pays** à attribuer.

## Étapes

1. Ouvrez **Admin** → **Gestion des utilisateurs et des accès** → **Gérer les utilisateurs**.
2. Cliquez sur **Ajouter un nouvel utilisateur**.
3. Renseignez les champs requis :
   - **E-mail** : identifiant de connexion
   - **Nom complet**
   - **Rôle** : Admin ou Point focal
   - **Mot de passe initial**
4. Si le rôle est **Point focal**, attribuez **au moins un pays**.
5. Cliquez sur **Créer l’utilisateur**.

## Après la création

- Partagez les identifiants de connexion **de manière sécurisée**.
- Si l’utilisateur ne voit pas les données attendues, vérifiez le **rôle** et les **pays attribués**.

## Conseils

- Les **admins** ont un accès étendu au système.
- Les **points focaux** doivent être limités aux pays concernés.
