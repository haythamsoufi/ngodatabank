# Revisar y aprobar envíos

Usa esta guía si eres responsable de verificaciones de calidad, aprobación o reapertura de asignaciones enviadas.

## Antes de comenzar

- Necesitas un rol apropiado (por ejemplo **Aprobador de asignaciones**).
- Saber qué significa "buena calidad" para tu organización (fuentes, totales, verificaciones de consistencia).

## Encontrar envíos para revisar

1. Abre **Panel de administración** → **Gestión de formularios y datos** → **Gestionar asignaciones**.
2. Haz clic en **Editar** en la asignación que quieres revisar.
3. En la sección Gestión de entidades, usa la información de estado (por ejemplo **enviado**) para encontrar qué países/entidades están listos para revisión.
4. Haz clic en un país/entidad para abrir el formulario de entrada para revisión.

## Revisar un envío

Cuando abres el formulario de un país enviado:

- Verifica que los campos requeridos estén completos.
- Busca valores atípicos obvios (valores inesperadamente altos/bajos).
- Verifica la consistencia interna (los totales suman, los campos relacionados se alinean).
- Revisa cualquier comentario o nota que el punto focal agregó (fuentes, explicaciones).

## Aprobar, rechazar o solicitar cambios

Dependiendo de lo que la interfaz muestre en tu sistema, puedes tener opciones como:

- **Aprobar**: confirma que el envío está aceptado.
- **Rechazar** (o "Devolver"): devuelve el envío para corrección.
- **Reabrir**: permite ediciones después del envío/aprobación (usa con moderación y registra por qué).

## Cuándo reabrir vs crear una nueva asignación

- **Reabrir** cuando es una corrección pequeña y quieres que el mismo envío sea corregido.
- **Crear una nueva asignación** cuando el cambio es grande, o la plantilla misma cambió, o necesitas un rastro de auditoría claro entre períodos.

## Consejos

- Escribe comentarios cortos y específicos: "Por favor corrige el campo X (sección Y). El total debe coincidir con Z."
- Acuerda de antemano una lista de verificación (3-10 elementos) para que las revisiones sean consistentes.

## Problemas comunes

- **No puedo aprobar**: confirma que tienes el rol **Aprobador de asignaciones**.
- **No puedo encontrar envíos**: confirma que la asignación está activa y los países han completado el envío.
- **Los datos cambiaron después de la aprobación**: evita reabrir a menos que sea necesario; documenta por qué reabriste.
