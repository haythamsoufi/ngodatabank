# إنشاء قالب النموذج

استخدم هذا الدليل لإنشاء قالب نموذج (الهيكل الذي ستُعينه لاحقاً للدول).

## قبل أن تبدأ

- تحتاج إلى وصول **المسؤول**.
- حدد المعلومات التي تحتاج إلى جمعها (الأسئلة، التنسيقات، روابط المؤشرات).

## الخطوات

1. افتح **لوحة المسؤول** → **إدارة النماذج والبيانات** → **إدارة القوالب**.
2. انقر على **إنشاء قالب**.
3. أدخل **اسم** القالب و(اختيارياً) **وصفاً**.
4. أضف **قسم** واحد أو أكثر لتنظيم النموذج.
5. داخل كل قسم، أضف **حقولاً** (نص، رقم، تاريخ، قائمة منسدلة، إلخ).
6. للحقول التي يجب أن تستخدم تعريفات قياسية، اربطها بـ **بنك المؤشرات** (إن كان قابلاً للتطبيق).
7. احفظ القالب.

## نصائح

- استخدم تسميات واضحة (النص الدقيق الذي سيراه المستخدمون).
- جمّع الأسئلة ذات الصلة في أقسام.
- حدد الحقول كـ **مطلوبة** فقط عند الضرورة.
- اختبر القالب بإنشاء مهمة صغيرة أولاً.

## المشاكل الشائعة

- **نقاط الاتصال مرتبكة بحقل**: أعد كتابة التسمية وأضف نص مساعدة قصير (إن كان متاحاً).
- **حقول مطلوبة كثيرة جداً**: المستخدمون يُحجَزون عند وقت الإرسال—حدد الحقول كمطلوبة فقط عند الحاجة الحقيقية.
