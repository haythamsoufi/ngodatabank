# Créer une nouvelle mission

Utilisez ce guide pour créer une nouvelle mission qui distribue un modèle de formulaire aux pays pour que les points focaux le remplissent.

## Avant de commencer

- Vous avez besoin d'un accès **Administrateur**.
- Vous avez besoin d'un **modèle** prêt avec une version publiée.
- Vous devriez savoir quels **pays** doivent recevoir la mission.

## Étapes

1. Ouvrez **Panneau d'administration** → **Gestion des formulaires et données** → **Gérer les missions**.
2. Cliquez sur **Créer une nouvelle mission**.
3. Sélectionnez le **modèle** que vous voulez distribuer (seuls les modèles avec des versions publiées apparaissent).
4. Entrez un **nom de période** :
   - Sélectionnez un **Type de période** : Année unique, Plage d'années ou Plage de mois
   - Entrez les valeurs appropriées en fonction du type de période sélectionné
   - Le système générera automatiquement le nom de la période
5. Définissez une **date limite** (date d'échéance) pour la soumission des données (optionnel mais recommandé).
6. Allez dans l'onglet **Missions basées sur les entités** pour sélectionner :
   - **Pays** qui doivent recevoir cette mission, et/ou
   - Entités de la **Structure NS** (branches, sous-branches, unités locales), et/ou
   - Entités du **Secrétariat** (divisions, départements, bureaux régionaux/clusters)
7. Si vous voulez autoriser les soumissions publiques sans connexion :
   - Cochez **Générer une URL publique pour cette mission**
   - Optionnellement, cochez **URL publique active par défaut** pour l'activer immédiatement
8. Cliquez sur **Créer** (ou **Enregistrer tout** si vous avez fait des sélections d'entités).

## Après avoir créé la mission

- Les points focaux assignés aux pays sélectionnés seront notifiés.
- Ils verront la nouvelle tâche dans leur tableau de bord.
- Vous pouvez surveiller la progression depuis la page Gérer les missions.

## Conseils

- Définissez des dates limites réalistes (les fuseaux horaires et les vacances comptent).
- Gardez les instructions courtes et spécifiques (ce qui est nécessaire, et quand).
- Choisissez un nom de période descriptif qui facilite l'identification de la mission plus tard.
- Commencez avec un petit ensemble de pays lors du test d'un nouveau modèle.
- Seuls les modèles avec des versions publiées peuvent être assignés.

## Problèmes courants

- **Impossible de créer la mission** : le modèle peut ne pas avoir de version publiée. Publiez d'abord le modèle.
- **Les points focaux ne voient pas la mission** : confirmez qu'ils sont assignés au pays et que la mission inclut ce pays.
