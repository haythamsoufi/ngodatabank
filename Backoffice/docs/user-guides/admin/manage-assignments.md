# Manage Assignments

Use this guide to view, edit, and manage existing assignments that distribute form templates to countries.

## Before you start

- You need **Admin** access.
- There should be at least one existing assignment in the system.

## View assignments

1. Open **Admin Panel** → **Form & Data Management** → **Manage Assignments**.
2. Review the list of all assignments.
3. See each assignment's period name, template, submission status, and public URL status.

## Monitor assignment progress

From the assignments list, you can:

- See which countries are **not started / in progress / submitted**
- Identify **overdue** submissions
- View submission progress by country
- Open an assignment to review details

## Edit an assignment

1. Find the assignment in the list.
2. Click the **edit** icon (pencil).
3. Update the fields you need:
   - **Template**: Change the form template (if needed)
   - **Period name**: Update the assignment name
   - **Deadline**: Extend or modify the submission deadline
4. To add more countries:
   - Use the **Add Countries** section
   - Select countries and click **Add**
5. Click **Save changes**.

## Manage public URLs

### Generate a public URL

1. Find the assignment in the list.
2. Click **Generate Public URL** (if not already generated).
3. The public URL allows submissions without login.

### Toggle public URL status

1. Find the assignment with a public URL.
2. Click the toggle to **activate** or **deactivate** the public URL.
3. When active, the public URL is accessible. When inactive, submissions are disabled.

### Copy public URL

1. Find the assignment with a public URL.
2. Click **Copy URL** to share it with external users.

## View public submissions

1. From the assignments page, click **View All Public Submissions** to see all public submissions.
2. Or click **View Submissions** for a specific assignment to see submissions for that assignment only.
3. You can approve, reject, or review submissions.

## Delete an assignment

1. Find the assignment in the list.
2. Click the **delete** icon (trash).
3. Confirm deletion.

**Warning**: This will remove the assignment and all associated country statuses and data. This action cannot be undone.

## Timeline view

1. From the assignments page, click **Timeline View**.
2. View all assignments on a timeline/Gantt chart to visualize deadlines and overlapping assignments.

## Tips

- Monitor the dashboard regularly for overdue submissions.
- Use the timeline view to avoid scheduling conflicts.
- Extend deadlines proactively if many countries are struggling.
- Public URLs are useful for external data collection but require monitoring.
- Review public submissions regularly to ensure data quality.

## Common problems

- **A focal point says they can't see the assignment**: confirm they are assigned to the country and the assignment includes that country.
- **Overdue submissions**: send a reminder and consider extending the deadline if needed.
- **Public URL not working**: check if it's activated and if the assignment is still active.
