# Indicator Bank (admin guide)

Use this guide if you manage standard indicator definitions and want consistent reporting across templates and assignments.

## What the Indicator Bank is

The **Indicator Bank** is a library of standardized indicators (names, definitions, and sometimes calculation rules). Linking form fields to indicators helps keep data consistent across:

- countries
- time periods
- different templates

## When to link a field to an indicator

Link a form field to an indicator when:

- you want the same measure reported the same way everywhere
- you plan to export and compare data across countries/periods
- you want a stable definition that doesn’t change with local wording

Do not link when:

- the field is purely operational text (notes, free-form explanations)
- the question is temporary or specific to one assignment

## Good practices

- Keep indicator definitions stable over time.
- If you must change meaning, create a new indicator rather than “reusing” the old one.
- Use consistent units (people, households, percentage, etc.).
- Make field labels user-friendly even if indicator names are technical.

## Troubleshooting

- **Exports look inconsistent**: confirm templates link to the correct indicators and that indicator definitions didn’t change mid-period.
- **Multiple fields linked to the same indicator**: check if they represent the same measure; if not, split into separate indicators.
