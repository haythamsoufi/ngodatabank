# Soumissions d'URL publiques

Utilisez ce guide lorsque vous voulez collecter des soumissions sans exiger que les utilisateurs se connectent.

## Avant de commencer

- Vous avez besoin d'un accès **Administrateur** pour gérer les missions.
- Comprenez le risque : les URL publiques peuvent être largement partagées. Utilisez-les uniquement lorsque c'est approprié.

## Quand utiliser (et quand ne pas utiliser)

Cas d'utilisation appropriés :
- Vous devez collecter des données de contributeurs qui ne peuvent pas obtenir de comptes à temps.
- Vous avez besoin d'un canal de capture de données à court terme pendant un événement urgent.

Évitez lorsque :
- Vous collectez des données sensibles.
- Vous avez besoin d'une responsabilité stricte (qui a soumis quoi).

## Créer ou activer une URL publique

1. Ouvrez **Panneau d'administration** → **Gestion des formulaires et données** → **Gérer les missions**.
2. Trouvez la mission.
3. Si la mission n'a pas encore d'URL publique, cliquez sur **Générer une URL publique**.
4. Utilisez le bouton pour **activer** l'URL publique.
5. Cliquez sur **Copier l'URL** pour la partager.

## Liste de contrôle de gouvernance (recommandé)

Avant de partager le lien :
- Décidez qui est autorisé à soumettre (et par quel canal vous distribuerez l'URL).
- Décidez comment vous gérerez les **doublons** (garder le plus récent / garder le meilleur / examen manuel).
- Décidez ce que signifie "qualité minimale" (champs requis, documents requis, preuves).
- Décidez quand vous désactiverez le lien (généralement juste après la date limite).

## Surveiller les soumissions publiques

1. Depuis **Gérer les missions**, utilisez :
   - **Voir toutes les soumissions publiques**, ou
   - **Voir les soumissions** pour une mission spécifique.
2. Examinez chaque soumission pour l'exhaustivité et la qualité.
3. Utilisez les actions disponibles (par exemple approuver/rejeter) pour gérer le contrôle qualité.

## Meilleures pratiques

- Partagez l'URL uniquement via des canaux de confiance (listes d'e-mails que vous contrôlez, plateformes officielles).
- Ajoutez des instructions claires dans le message de la mission (quoi remplir, date limite, qui contacter).
- Surveillez les soumissions régulièrement pendant la période de collecte.
- Décidez comment gérer les doublons (par exemple : garder le plus récent, ou garder celui avec les meilleures sources).
- Désactivez l'URL publique lorsque la fenêtre de collecte se termine (pour réduire les risques).
- Si vous soupçonnez que le lien a été partagé trop largement, désactivez-le immédiatement et créez un nouveau canal contrôlé.

## Problèmes courants

- **L'URL publique "ne fonctionne pas"** : confirmez qu'elle est activée et que la mission est toujours active.
- **Trop de soumissions incorrectes** : ajoutez des instructions plus claires, réduisez l'ambiguïté et envisagez de désactiver l'URL publique.
- **Soumissions en double** : décidez de votre règle (garder le plus récent / garder le meilleur / fusion manuelle) et appliquez-la de manière cohérente.
