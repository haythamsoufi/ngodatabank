# Cycle de vie d'une mission (de la planification à la clôture)

Utilisez ce guide pour gérer une mission dans le temps : avant qu'elle ne commence, pendant la collecte et après la soumission.

## Avant de commencer

- Vous avez besoin d'un accès **Administrateur**.
- Confirmez que le modèle est prêt et testé.
- Décidez du calendrier (date de début, date limite, fenêtre d'examen).

## Planifier (avant le lancement)

- Choisissez un **nom de période** clair (pour que les pays le reconnaissent).
- Écrivez un court message d'instructions :
  - Ce qui est demandé
  - Date limite
  - Qui contacter pour les questions
- Décidez si vous voulez utiliser une **URL publique** (seulement si nécessaire).

## Lancement

1. Créez la mission.
2. Confirmez que les bons pays sont inclus.
3. Confirmez que les points focaux pour ces pays ont :
   - les rôles de mission corrects, et
   - l'accès au pays correct.

## Surveiller pendant la collecte

Pendant la période ouverte, vérifiez régulièrement la progression :

- Pays non commencés
- Pays en cours
- Pays soumis
- Pays en retard

Si de nombreux pays ont des difficultés :

- Prolongez la date limite (si autorisé dans votre flux de travail).
- Clarifiez les questions confuses dans le modèle (libellé/texte d'aide).

## Gérer les changements après le lancement

Les changements typiques incluent :

- **Ajouter des pays** : ajoutez-les dans l'écran de modification de la mission, puis notifiez les points focaux.
- **Prolonger la date limite** : communiquez clairement la nouvelle date limite.
- **Modifications du modèle** : évitez les grandes modifications du modèle en milieu de collecte ; si nécessaire, publiez une nouvelle version et créez une nouvelle mission (préféré).

## Examiner et fermer

Après l'arrivée des soumissions :

- Examinez la qualité (cohérence, valeurs aberrantes, champs requis manquants).
- Approuvez ou rouvrez les soumissions selon les besoins.
- Exportez les données pour les rapports.

## Problèmes courants

- **Un point focal ne peut pas voir la mission** : vérifiez d'abord ses rôles et son accès au pays.
- **Taux de soumission faible** : envisagez des rappels, des changements de date limite et des modèles plus simples.
- **Champs de modèle confus** : corrigez les libellés/texte d'aide et réduisez les champs requis.
