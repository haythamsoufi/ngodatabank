# إضافة مستخدم جديد

استخدم هذا الدليل لإنشاء حساب جديد في Backoffice لمسؤول أو لنقطة اتصال.

## قبل البدء

- تحتاج إلى صلاحية **مسؤول (Admin)**.
- إذا كان المستخدم **نقطة اتصال**، يجب معرفة **الدولة/الدول** التي سيتم تعيينها له.

## الخطوات

1. افتح **Admin** → **إدارة المستخدمين والصلاحيات** → **إدارة المستخدمين**.
2. اضغط **إضافة مستخدم جديد**.
3. املأ الحقول المطلوبة:
   - **البريد الإلكتروني**: سيكون اسم المستخدم لتسجيل الدخول
   - **الاسم الكامل**
   - **الدور**: مسؤول أو نقطة اتصال
   - **كلمة مرور أولية**
4. إذا كان الدور **نقطة اتصال**، قم بتعيين **دولة واحدة على الأقل**.
5. اضغط **إنشاء المستخدم**.

## بعد الإنشاء

- شارك بيانات تسجيل الدخول **بشكل آمن**.
- إذا لم يتمكن المستخدم من رؤية البيانات المطلوبة، راجع **الدور** و**الدول المعيّنة**.

## نصائح

- لدى **المسؤولين** صلاحيات واسعة على النظام.
- يجب أن تقتصر صلاحيات **نقاط الاتصال** على الدول المعنية.
