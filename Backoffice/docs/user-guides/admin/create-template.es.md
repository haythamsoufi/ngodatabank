# Crear plantilla de formulario

Usa esta guía para crear una plantilla de formulario (la estructura que luego asignarás a países).

## Antes de comenzar

- Necesitas acceso de **Administrador**.
- Aclara qué información necesitas recopilar (preguntas, formatos, enlaces de indicadores).

## Pasos

1. Abre **Panel de administración** → **Gestión de formularios y datos** → **Gestionar plantillas**.
2. Haz clic en **Crear plantilla**.
3. Ingresa el **nombre** de la plantilla y (opcionalmente) una **descripción**.
4. Agrega una o más **secciones** para organizar el formulario.
5. Dentro de cada sección, agrega **campos** (texto, número, fecha, lista desplegable, etc.).
6. Para campos que deben usar definiciones estándar, enlázalos al **Banco de indicadores** (si aplica).
7. Guarda la plantilla.

## Consejos

- Usa etiquetas claras (el texto exacto que verán los usuarios).
- Agrupa preguntas relacionadas en secciones.
- Marca campos como **requeridos** solo cuando sea necesario.
- Prueba la plantilla creando primero una pequeña asignación.

## Problemas comunes

- **Los puntos focales están confundidos por un campo**: reescribe la etiqueta y agrega un texto de ayuda corto (si está disponible).
- **Demasiados campos requeridos**: los usuarios se bloquean al enviar—marca campos como requeridos solo cuando sea realmente necesario.
