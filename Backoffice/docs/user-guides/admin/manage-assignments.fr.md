# Gérer les missions

Utilisez ce guide pour voir, modifier et gérer les missions existantes qui distribuent des modèles de formulaires aux pays.

## Avant de commencer

- Vous avez besoin d'un accès **Administrateur**.
- Il devrait y avoir au moins une mission existante dans le système.

## Voir les missions

1. Ouvrez **Panneau d'administration** → **Gestion des formulaires et données** → **Gérer les missions**.
2. Examinez la liste de toutes les missions.
3. Voyez le nom de période, le modèle, le statut de soumission et le statut de l'URL publique de chaque mission.

## Surveiller la progression des missions

Depuis la liste des missions, vous pouvez :

- Voir quels pays sont **non commencés / en cours / soumis**
- Identifier les soumissions **en retard**
- Voir la progression des soumissions par pays
- Ouvrir une mission pour examiner les détails

## Modifier une mission

1. Trouvez la mission dans la liste.
2. Cliquez sur l'icône **modifier** (crayon).
3. Mettez à jour les champs dont vous avez besoin :
   - **Modèle** : Changez le modèle de formulaire (si nécessaire)
   - **Nom de période** : Mettez à jour le nom de la mission
   - **Date limite** : Prolongez ou modifiez la date limite de soumission
4. Pour ajouter plus de pays :
   - Utilisez la section **Ajouter des pays**
   - Sélectionnez les pays et cliquez sur **Ajouter**
5. Cliquez sur **Enregistrer les modifications**.

## Gérer les URL publiques

### Générer une URL publique

1. Trouvez la mission dans la liste.
2. Cliquez sur **Générer une URL publique** (si elle n'est pas déjà générée).
3. L'URL publique permet les soumissions sans connexion.

### Basculer le statut de l'URL publique

1. Trouvez la mission avec une URL publique.
2. Cliquez sur le bouton pour **activer** ou **désactiver** l'URL publique.
3. Lorsqu'elle est active, l'URL publique est accessible. Lorsqu'elle est inactive, les soumissions sont désactivées.

### Copier l'URL publique

1. Trouvez la mission avec une URL publique.
2. Cliquez sur **Copier l'URL** pour la partager avec des utilisateurs externes.

## Voir les soumissions publiques

1. Depuis la page des missions, cliquez sur **Voir toutes les soumissions publiques** pour voir toutes les soumissions publiques.
2. Ou cliquez sur **Voir les soumissions** pour une mission spécifique pour voir les soumissions pour cette mission uniquement.
3. Vous pouvez approuver, rejeter ou examiner les soumissions.

## Supprimer une mission

1. Trouvez la mission dans la liste.
2. Cliquez sur l'icône **supprimer** (corbeille).
3. Confirmez la suppression.

**Avertissement** : Cela supprimera la mission et tous les statuts de pays et données associés. Cette action ne peut pas être annulée.

## Vue chronologique

1. Depuis la page des missions, cliquez sur **Vue chronologique**.
2. Voyez toutes les missions sur un graphique chronologique/Gantt pour visualiser les dates limites et les missions qui se chevauchent.

## Conseils

- Surveillez régulièrement le tableau de bord pour les soumissions en retard.
- Utilisez la vue chronologique pour éviter les conflits de planification.
- Prolongez les dates limites de manière proactive si de nombreux pays ont des difficultés.
- Les URL publiques sont utiles pour la collecte de données externes mais nécessitent une surveillance.
- Examinez régulièrement les soumissions publiques pour assurer la qualité des données.

## Problèmes courants

- **Un point focal dit qu'il ne peut pas voir la mission** : confirmez qu'il est assigné au pays et que la mission inclut ce pays.
- **Soumissions en retard** : envoyez un rappel et envisagez de prolonger la date limite si nécessaire.
- **L'URL publique ne fonctionne pas** : vérifiez si elle est activée et si la mission est toujours active.
