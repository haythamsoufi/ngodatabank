# Crear nueva asignación

Usa esta guía para crear una nueva asignación que distribuya una plantilla de formulario a países para que los puntos focales la completen.

## Antes de comenzar

- Necesitas acceso de **Administrador**.
- Necesitas una **plantilla** lista con una versión publicada.
- Debes saber qué **países** deben recibir la asignación.

## Pasos

1. Abre **Panel de administración** → **Gestión de formularios y datos** → **Gestionar asignaciones**.
2. Haz clic en **Crear nueva asignación**.
3. Selecciona la **plantilla** que quieres distribuir (solo aparecen plantillas con versiones publicadas).
4. Ingresa un **nombre de período**:
   - Selecciona un **Tipo de período**: Año único, Rango de años o Rango de meses
   - Ingresa los valores apropiados según el tipo de período seleccionado
   - El sistema generará el nombre del período automáticamente
5. Establece una **fecha límite** (fecha de vencimiento) para el envío de datos (opcional pero recomendado).
6. Ve a la pestaña **Asignaciones basadas en entidades** para seleccionar:
   - **Países** que deben recibir esta asignación, y/o
   - Entidades de la **Estructura NS** (sucursales, sub-sucursales, unidades locales), y/o
   - Entidades del **Secretariado** (divisiones, departamentos, oficinas regionales/cluster)
7. Si quieres permitir envíos públicos sin inicio de sesión:
   - Marca **Generar URL pública para esta asignación**
   - Opcionalmente marca **URL pública activa por defecto** para activarla inmediatamente
8. Haz clic en **Crear** (o **Guardar todo** si has hecho selecciones de entidades).

## Después de crear la asignación

- Los puntos focales asignados a los países seleccionados serán notificados.
- Verán la nueva tarea en su panel de control.
- Puedes monitorear el progreso desde la página Gestionar asignaciones.

## Consejos

- Establece fechas límite realistas (las zonas horarias y los días festivos importan).
- Mantén las instrucciones cortas y específicas (qué se necesita y para cuándo).
- Elige un nombre de período descriptivo que facilite identificar la asignación más tarde.
- Comienza con un pequeño conjunto de países al probar una nueva plantilla.
- Solo las plantillas con versiones publicadas pueden ser asignadas.

## Problemas comunes

- **No se puede crear la asignación**: la plantilla puede no tener una versión publicada. Publica la plantilla primero.
- **Los puntos focales no ven la asignación**: confirma que están asignados al país y que la asignación incluye ese país.
