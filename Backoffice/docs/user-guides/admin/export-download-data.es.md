# Exportar y descargar datos

Usa esta guía para descargar datos de asignación para análisis o reportes (por ejemplo a Excel o PDF).

## Antes de comenzar

- Necesitas acceso a la asignación que quieres exportar.
- La plantilla debe tener las funciones de exportación habilitadas (exportación Excel y/o PDF).
- La exportación está disponible desde la **página del formulario de entrada** al completar o ver una asignación.

## Exportar datos

### Exportar desde la página del formulario de entrada

Las opciones de exportación están disponibles cuando estás en la página del formulario de entrada de la asignación (completando o viendo el formulario):

1. **Navegar al formulario de asignación**:
   - Desde tu **Panel de control**, abre la asignación que quieres exportar.
   - O, si eres administrador: Abre **Panel de administración** → **Gestión de formularios y datos** → **Gestionar asignaciones**, luego haz clic en un país/entidad en la asignación para abrir el formulario de entrada.

2. **Exportar plantilla Excel** (si está habilitado):
   - Haz clic en el botón **Opciones Excel** (o elemento de menú) en la interfaz del formulario.
   - Haz clic en **Exportar plantilla de datos (.xlsx)** para descargar un archivo Excel con la estructura del formulario y los datos actuales.
   - El archivo Excel se puede usar para completar datos sin conexión y luego importarlo de nuevo.

3. **Exportar PDF** (si está habilitado):
   - Haz clic en el botón **Exportar PDF** en la interfaz del formulario.
   - Se generará y descargará una versión PDF del formulario con los datos actuales.
   - Esto es útil para imprimir o archivar envíos.

**Nota**: Las funciones de exportación deben estar habilitadas en la plantilla por un administrador. Si no ves opciones de exportación, la plantilla puede no tener estas funciones habilitadas.

## Consejos

- Mantén siempre el nombre del archivo exportado consistente (período + plantilla + alcance).
- Si comparas entre períodos, mantén la misma versión de plantilla cuando sea posible.
- Si la exportación incluye ID/códigos, mantenlos (ayudan a unir conjuntos de datos más tarde).

## Problemas comunes

- **La exportación falta un país**: el país puede no ser parte de la asignación, o aún no ha enviado, o tu filtro de exportación lo está excluyendo.
- **Los números no coinciden con el formulario**: verifica si exportaste una versión de plantilla/período diferente, o si se aplican cálculos/redondeo en las exportaciones.
- **La exportación tarda demasiado**: exporta alcances más pequeños (una asignación a la vez) o menos países.
