# Agregar Nuevo Usuario

Use esta guía para crear una nueva cuenta de Backoffice para un administrador o un punto focal.

## Antes de comenzar

- Necesita acceso de **Admin**.
- Para un **punto focal**, debe saber qué **país(es)** se deben asignar.

## Pasos

1. Abra **Admin** → **Gestión de usuarios y accesos** → **Gestionar usuarios**.
2. Haga clic en **Agregar nuevo usuario**.
3. Complete los campos requeridos:
   - **Correo**: será el usuario de inicio de sesión
   - **Nombre completo**
   - **Rol**: Admin o Punto focal
   - **Contraseña inicial**
4. Si el rol es **Punto focal**, asigne **al menos un país**.
5. Haga clic en **Crear usuario**.

## Después de crear el usuario

- Comparta las credenciales **de forma segura**.
- Si el usuario no ve los datos esperados, revise el **rol** y los **países asignados**.

## Consejos

- Los **admins** tienen acceso amplio al sistema.
- Los **puntos focales** deben limitarse a los países correspondientes.
