# Examiner et approuver les soumissions

Utilisez ce guide si vous êtes responsable des vérifications de qualité, de l'approbation ou de la réouverture des missions soumises.

## Avant de commencer

- Vous avez besoin d'un rôle approprié (par exemple **Approbateur de mission**).
- Sachez ce que signifie "bonne qualité" pour votre organisation (sources, totaux, vérifications de cohérence).

## Trouver les soumissions à examiner

1. Ouvrez **Panneau d'administration** → **Gestion des formulaires et données** → **Gérer les missions**.
2. Cliquez sur **Modifier** sur la mission que vous voulez examiner.
3. Dans la section Gestion des entités, utilisez les informations de statut (par exemple **soumis**) pour trouver quels pays/entités sont prêts pour examen.
4. Cliquez sur un pays/entité pour ouvrir le formulaire d'entrée pour examen.

## Examiner une soumission

Lorsque vous ouvrez le formulaire d'un pays soumis :

- Vérifiez que les champs requis sont remplis.
- Recherchez les valeurs aberrantes évidentes (valeurs anormalement élevées/faibles).
- Vérifiez la cohérence interne (les totaux s'additionnent, les champs connexes s'alignent).
- Examinez tous les commentaires ou notes que le point focal a ajoutés (sources, explications).

## Approuver, rejeter ou demander des modifications

Selon ce que l'interface affiche dans votre système, vous pouvez avoir des options comme :

- **Approuver** : confirme que la soumission est acceptée.
- **Rejeter** (ou "Renvoi") : renvoie la soumission pour correction.
- **Rouvrir** : permet les modifications après soumission/approbation (utilisez avec parcimonie et suivez pourquoi).

## Quand rouvrir vs créer une nouvelle mission

- **Rouvrir** lorsqu'il s'agit d'une petite correction et que vous voulez que la même soumission soit corrigée.
- **Créer une nouvelle mission** lorsque le changement est important, ou que le modèle lui-même a changé, ou que vous avez besoin d'une piste d'audit claire entre les périodes.

## Conseils

- Écrivez des commentaires courts et spécifiques : "Veuillez corriger le champ X (section Y). Le total doit correspondre à Z."
- Convenez à l'avance d'une liste de contrôle (3-10 éléments) pour que les examens soient cohérents.

## Problèmes courants

- **Je ne peux pas approuver** : confirmez que vous avez le rôle **Approbateur de mission**.
- **Je ne trouve pas les soumissions** : confirmez que la mission est active et que les pays ont terminé la soumission.
- **Les données ont changé après approbation** : évitez de rouvrir sauf si nécessaire ; documentez pourquoi vous avez rouvert.
