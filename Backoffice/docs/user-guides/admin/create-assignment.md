# Create New Assignment

Use this guide to create a new assignment that distributes a form template to countries for focal points to fill in.

## Before you start

- You need **Admin** access.
- You need a ready **template** with a published version.
- You should know which **countries** should receive the assignment.

## Steps

1. Open **Admin Panel** → **Form & Data Management** → **Manage Assignments**.
2. Click **Create New Assignment**.
3. Select the **template** you want to distribute (only templates with published versions appear).
4. Enter a **period name**:
   - Select a **Period Type**: Single Year, Year Range, or Month Range
   - Enter the appropriate values based on the period type selected
   - The system will generate the period name automatically
5. Set a **deadline** (due date) for data submission (optional but recommended).
6. Go to the **Entity-Based Assignments** tab to select:
   - **Countries** that should receive this assignment, and/or
   - **NS Structure** entities (branches, sub-branches, local units), and/or
   - **Secretariat** entities (divisions, departments, regional/cluster offices)
7. If you want to allow public submissions without login:
   - Check **Generate public URL for this assignment**
   - Optionally check **Public URL active by default** to activate it immediately
8. Click **Create** (or **Save All** if you've made entity selections).

## After you create the assignment

- Focal points assigned to the selected countries will be notified.
- They will see the new task in their dashboard.
- You can monitor progress from the Manage Assignments page.

## Tips

- Set realistic deadlines (time zones and holidays matter).
- Keep instructions short and specific (what's needed, and by when).
- Choose a descriptive period name that makes it easy to identify the assignment later.
- Start with a small set of countries when testing a new template.
- Only templates with published versions can be assigned.

## Common problems

- **Cannot create assignment**: the template may not have a published version. Publish the template first.
- **Focal points don't see the assignment**: confirm they are assigned to the country and the assignment includes that country.
