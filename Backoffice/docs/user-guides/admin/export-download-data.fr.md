# Exporter et télécharger les données

Utilisez ce guide pour télécharger les données de mission pour analyse ou reporting (par exemple vers Excel ou PDF).

## Avant de commencer

- Vous avez besoin d'accès à la mission que vous voulez exporter.
- Le modèle doit avoir les fonctionnalités d'export activées (export Excel et/ou PDF).
- L'export est disponible depuis la **page du formulaire d'entrée** lors du remplissage ou de la visualisation d'une mission.

## Exporter les données

### Exporter depuis la page du formulaire d'entrée

Les options d'export sont disponibles lorsque vous êtes sur la page du formulaire d'entrée de la mission (remplissage ou visualisation du formulaire) :

1. **Naviguer vers le formulaire de mission** :
   - Depuis votre **Tableau de bord**, ouvrez la mission que vous voulez exporter.
   - Ou, si vous êtes administrateur : Ouvrez **Panneau d'administration** → **Gestion des formulaires et données** → **Gérer les missions**, puis cliquez sur un pays/entité dans la mission pour ouvrir le formulaire d'entrée.

2. **Exporter le modèle Excel** (si activé) :
   - Cliquez sur le bouton **Options Excel** (ou élément de menu) dans l'interface du formulaire.
   - Cliquez sur **Exporter le modèle de données (.xlsx)** pour télécharger un fichier Excel avec la structure du formulaire et les données actuelles.
   - Le fichier Excel peut être utilisé pour remplir les données hors ligne puis importé à nouveau.

3. **Exporter PDF** (si activé) :
   - Cliquez sur le bouton **Exporter PDF** dans l'interface du formulaire.
   - Une version PDF du formulaire avec les données actuelles sera générée et téléchargée.
   - C'est utile pour imprimer ou archiver les soumissions.

**Note** : Les fonctionnalités d'export doivent être activées sur le modèle par un administrateur. Si vous ne voyez pas les options d'export, le modèle peut ne pas avoir ces fonctionnalités activées.

## Conseils

- Gardez toujours le nom du fichier exporté cohérent (période + modèle + portée).
- Si vous comparez entre les périodes, gardez la même version du modèle lorsque c'est possible.
- Si l'export inclut des ID/codes, gardez-les (ils aident à joindre les ensembles de données plus tard).

## Problèmes courants

- **L'export manque un pays** : le pays peut ne pas faire partie de la mission, ou il n'a pas encore soumis, ou votre filtre d'export l'exclut.
- **Les chiffres ne correspondent pas au formulaire** : vérifiez si vous avez exporté une version de modèle/période différente, ou si des calculs/arrondis sont appliqués dans les exports.
- **L'export prend trop de temps** : exportez des portées plus petites (une mission à la fois) ou moins de pays.
