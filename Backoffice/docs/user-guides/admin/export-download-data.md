# Export and download data

Use this guide to download assignment data for analysis or reporting (for example to Excel or PDF).

## Before you start

- You need access to the assignment you want to export.
- The template must have export features enabled (Excel and/or PDF export).
- Export is available from the **entry form page** when filling out or viewing an assignment.

## Export data

### Export from the entry form page

Export options are available when you are on the assignment entry form page (filling out or viewing the form):

1. **Navigate to the assignment form**:
   - From your **Dashboard**, open the assignment you want to export.
   - Or, if you're an admin: Open **Admin Panel** → **Form & Data Management** → **Manage Assignments**, then click on a country/entity in the assignment to open the entry form.

2. **Export Excel template** (if enabled):
   - Click the **Excel Options** button (or menu item) in the form interface.
   - Click **Export Data Template (.xlsx)** to download an Excel file with the form structure and current data.
   - The Excel file can be used to fill in data offline and then imported back.

3. **Export PDF** (if enabled):
   - Click the **Export PDF** button in the form interface.
   - A PDF version of the form with current data will be generated and downloaded.
   - This is useful for printing or archiving submissions.

**Note**: Export features must be enabled on the template by an administrator. If you don't see export options, the template may not have these features enabled.

## Tips

- Always keep the exported file name consistent (period + template + scope).
- If you compare across periods, keep the same template version when possible.
- If the export includes IDs/codes, keep them (they help join datasets later).

## Common problems

- **Export is missing a country**: the country might not be part of the assignment, or it hasn’t submitted yet, or your export filter is excluding it.
- **Numbers don’t match the form**: check if you exported a different template version/period, or if calculations/rounding are applied in exports.
- **Export takes too long**: export smaller scopes (one assignment at a time) or fewer countries.
