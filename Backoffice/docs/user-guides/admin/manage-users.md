# Manage users

Use this guide to find users, update their details, and manage access.

## Before you start

- You need **Admin** access.

## Find a user

1. Open **Admin Panel** → **User Management** → **Manage Users**.
2. Use the search box to find the user by **name** or **email**.
3. Open the user to view or edit details.

## Update user details

1. Open the user.
2. In the **User Details** tab, update the fields you need (for example: name, title, password).
3. To update roles, select or deselect roles in the Roles section (users can have multiple roles).
4. Click **Save changes**.

## Assign or remove countries/organizations (entity permissions)

1. Open the user.
2. Go to the **Entity Permissions** tab.
3. Add or remove **countries** or **organizations** (NS branches, divisions, etc.).
4. Click **Save changes**.

## Deactivate a user

1. Find the user in the list.
2. Click **Deactivate** (or toggle the user status).
3. Confirm.

Deactivated users cannot log in, but their history is preserved.

## Reset a password

If your workflow requires setting a new password for a user:

1. Open the user.
2. Set a new password.
3. Save.
4. Share the new password **securely**.

## Tips

- If a focal point reports missing data, check **country assignments** first.
- Prefer **deactivating** accounts instead of deleting them.

## Common problems

- **User cannot see assignments**: confirm they are a focal point and have at least one country assigned.
- **User cannot access an admin page**: confirm their role/permissions include that page.
