# Create Form Template

Use this guide to create a form template (the structure you will later assign to countries).

## Before you start

- You need **Admin** access.
- Clarify what information you need to collect (questions, formats, indicator links).

## Steps

1. Open **Admin Panel** → **Form & Data Management** → **Manage Templates**.
2. Click **Create template**.
3. Enter the template **name** and (optionally) a **description**.
4. Add one or more **sections** to organize the form.
5. Inside each section, add **fields** (text, number, date, dropdown, etc.).
6. For fields that should use standard definitions, link them to the **Indicator bank** (if applicable).
7. Save the template.

## Tips

- Use clear labels (the exact text users will see).
- Group related questions into sections.
- Mark fields as **required** only when necessary.
- Test the template by creating a small assignment first.

## Common problems

- **Focal points are confused by a field**: rewrite the label and add a short help text (if available).
- **Too many required fields**: users get blocked at submit time—mark fields required only when truly needed.
