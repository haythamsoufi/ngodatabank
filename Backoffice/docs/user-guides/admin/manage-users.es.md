# Gestionar usuarios

Usa esta guía para encontrar usuarios, actualizar sus detalles y gestionar el acceso.

## Antes de comenzar

- Necesitas acceso de **Administrador**.

## Encontrar un usuario

1. Abre **Panel de administración** → **Gestión de usuarios** → **Gestionar usuarios**.
2. Usa el cuadro de búsqueda para encontrar el usuario por **nombre** o **correo electrónico**.
3. Abre el usuario para ver o editar detalles.

## Actualizar detalles del usuario

1. Abre el usuario.
2. En la pestaña **Detalles del usuario**, actualiza los campos que necesites (por ejemplo: nombre, título, contraseña).
3. Para actualizar roles, selecciona o deselecciona roles en la sección Roles (los usuarios pueden tener múltiples roles).
4. Haz clic en **Guardar cambios**.

## Asignar o eliminar países/organizaciones (permisos de entidad)

1. Abre el usuario.
2. Ve a la pestaña **Permisos de entidad**.
3. Agrega o elimina **países** u **organizaciones** (sucursales NS, divisiones, etc.).
4. Haz clic en **Guardar cambios**.

## Desactivar un usuario

1. Encuentra el usuario en la lista.
2. Haz clic en **Desactivar** (o cambia el estado del usuario).
3. Confirma.

Los usuarios desactivados no pueden iniciar sesión, pero su historial se conserva.

## Restablecer una contraseña

Si tu flujo de trabajo requiere establecer una nueva contraseña para un usuario:

1. Abre el usuario.
2. Establece una nueva contraseña.
3. Guarda.
4. Comparte la nueva contraseña **de forma segura**.

## Consejos

- Si un punto focal reporta datos faltantes, verifica primero las **asignaciones de países**.
- Prefiere **desactivar** cuentas en lugar de eliminarlas.

## Problemas comunes

- **El usuario no puede ver asignaciones**: confirma que es un punto focal y tiene al menos un país asignado.
- **El usuario no puede acceder a una página de administración**: confirma que su rol/permisos incluyen esa página.
