# Documents justificatifs (administrateur)

Utilisez ce guide pour comprendre comment les documents justificatifs sont liés aux missions (par exemple : fichiers de preuve, rapports ou pièces jointes).

## Ce que sont les documents justificatifs

Les documents justificatifs sont des fichiers joints à une soumission de mission pour fournir :

- des documents de preuve ou de source
- des rapports de contexte
- des feuilles de calcul utilisées pour calculer les valeurs

## Ce qu'il faut exiger (gardez-le minimal)

Si vous décidez d'exiger des documents :
- Exigez l'ensemble minimum nécessaire pour valider la soumission.
- Soyez explicite : quel fichier, quel format et quelle convention de nommage.

Exemple de texte d'exigence :
- "Téléchargez un PDF avec la source officielle pour les totaux rapportés (nommé `Country_Period_Source.pdf`)."

## Où trouver les documents

Selon votre configuration, vous pouvez trouver des documents :

- dans la vue détaillée d'une mission, ou
- dans une zone "Documents" pour la mission/soumission

## Examiner les documents

Lors de l'examen d'une soumission :

1. Ouvrez la soumission.
2. Vérifiez que les documents joints existent lorsque votre processus l'exige.
3. Confirmez que le document correspond aux valeurs rapportées (l'échantillonnage est acceptable si le volume est élevé).

## Gestion des données sensibles (important)

- Évitez de demander des documents avec des identifiants personnels sauf si c'est explicitement requis et approuvé.
- Si des documents sensibles sont requis, convenez de :
  - qui peut y accéder
  - où ils peuvent être exportés/partagés
  - combien de temps ils sont conservés

## Bonnes pratiques

- Convenez de conventions de nommage (exemple : `Country_Period_Source.pdf`).
- Préférez un petit ensemble de documents requis plutôt que de nombreux fichiers optionnels.
- Évitez de télécharger des données personnelles sensibles sauf si c'est explicitement requis et approuvé.

## Problèmes courants

- **Un utilisateur ne peut pas télécharger de documents** : confirmez qu'il a un rôle qui autorise le téléchargement de documents (par exemple `assignment_documents_uploader` ou `assignment_editor_submitter`).
- **Les documents sont manquants** : clarifiez si les documents sont requis pour cette mission et communiquez les attentes aux points focaux.
