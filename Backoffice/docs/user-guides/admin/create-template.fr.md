# Créer un modèle de formulaire

Utilisez ce guide pour créer un modèle de formulaire (la structure que vous assignerez plus tard aux pays).

## Avant de commencer

- Vous avez besoin d'un accès **Administrateur**.
- Clarifiez les informations que vous devez collecter (questions, formats, liens d'indicateurs).

## Étapes

1. Ouvrez **Panneau d'administration** → **Gestion des formulaires et données** → **Gérer les modèles**.
2. Cliquez sur **Créer un modèle**.
3. Entrez le **nom** du modèle et (optionnellement) une **description**.
4. Ajoutez une ou plusieurs **sections** pour organiser le formulaire.
5. Dans chaque section, ajoutez des **champs** (texte, nombre, date, liste déroulante, etc.).
6. Pour les champs qui doivent utiliser des définitions standard, liez-les à la **Banque d'indicateurs** (si applicable).
7. Enregistrez le modèle.

## Conseils

- Utilisez des libellés clairs (le texte exact que les utilisateurs verront).
- Groupez les questions connexes en sections.
- Marquez les champs comme **requis** uniquement lorsque c'est nécessaire.
- Testez le modèle en créant d'abord une petite mission.

## Problèmes courants

- **Les points focaux sont confus par un champ** : réécrivez le libellé et ajoutez un court texte d'aide (si disponible).
- **Trop de champs requis** : les utilisateurs sont bloqués au moment de la soumission—marquez les champs comme requis uniquement lorsque c'est vraiment nécessaire.
