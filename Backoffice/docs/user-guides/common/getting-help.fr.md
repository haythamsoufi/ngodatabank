# Obtenir de l'aide

Utilisez cette page si vous êtes bloqué et avez besoin d'aide.

## D'abord, essayez ces vérifications rapides

- **Actualisez la page** et réessayez (des problèmes temporaires peuvent survenir).
- Assurez-vous d'être dans le **bon pays** (si votre travail dépend de l'accès au pays).
- Vérifiez votre **connexion Internet**.

## Si vous ne voyez pas quelque chose que vous attendiez

- Si vous êtes un point focal et **ne voyez pas de missions**, vous n'êtes peut-être pas encore assigné à un pays.
- Si vous pouvez vous connecter mais **ne voyez pas les bons éléments de menu**, votre rôle/permissions peut être limité.

## Comment demander de l'aide (informations recommandées)

Lorsque vous contactez le support, incluez :

- Ce que vous essayiez de faire (1-2 phrases)
- La page sur laquelle vous étiez (copiez l'URL si possible)
- Votre rôle (administrateur ou point focal)
- Votre/vos pays (si pertinent)
- Une capture d'écran du message/erreur
- L'heure à laquelle cela s'est produit
