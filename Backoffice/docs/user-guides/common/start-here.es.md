# Comienza aquí

Usa esta página si eres nuevo y quieres entender qué hacer primero.

## Inicio rápido

1. Abre tu **Panel de control**.
2. Establece tu **Idioma** en [Configuración de cuenta](account-settings.md).
3. Abre [Ver tus asignaciones](../focal-point/view-assignments.md) para encontrar los formularios que debes completar.
