# Configuración de cuenta

Usa esta página para actualizar tu perfil y preferencias.

## Qué puedes actualizar

- **Perfil**: nombre, título (título del trabajo/rol)
- **Color del perfil**: elige un color para el icono de tu perfil
- **Chatbot de IA**: activa o desactiva el asistente de chatbot de IA (si está disponible)
- **Dispositivos registrados**: ver y gestionar tus dispositivos móviles registrados para notificaciones push

## Qué no puedes cambiar aquí

- **Dirección de correo electrónico**: Tu correo electrónico es gestionado por SSO y no se puede cambiar. Contacta a tu administrador del sistema.
- **Contraseña**: Los cambios de contraseña deben ser realizados por un administrador del sistema. Contacta a tu administrador para cambiar tu contraseña.
- **Roles**: Tus roles son gestionados por administradores del sistema.

## Actualizar tu perfil

1. Abre **Configuración de cuenta** (desde tu panel de control o menú de usuario).
2. Actualiza tu **Nombre** y **Título** (opcional).
3. Elige un **Color de perfil** para el icono de tu perfil.
4. Si está disponible, activa o desactiva **Chatbot de IA**.
5. Haz clic en **Guardar cambios**.

## Gestionar dispositivos registrados

Puedes ver y gestionar tus dispositivos móviles registrados:

- **Ver dispositivos**: Ver todos los dispositivos registrados para notificaciones push
- **Finalizar sesión**: Cerrar sesión desde un dispositivo específico (mantiene el dispositivo registrado)
- **Eliminar dispositivo**: Eliminar permanentemente un dispositivo del registro

**Nota**: Para cambiar tu dirección de correo electrónico o contraseña, por favor contacta a tu administrador del sistema.

## Consejos

- Si cambias tu idioma, algunas páginas pueden seguir apareciendo en inglés hasta que se traduzcan.
- Mantén tu información de contacto actualizada.
