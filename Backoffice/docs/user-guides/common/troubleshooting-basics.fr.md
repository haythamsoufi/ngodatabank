# Bases du dépannage (tout le monde)

Utilisez cette page pour des corrections rapides avant d'escalader vers une équipe d'administration/support.

## Premières vérifications (rapides)

- Actualisez la page.
- Déconnectez-vous et reconnectez-vous.
- Essayez un autre navigateur (si possible).
- Si vous êtes sur une connexion lente, attendez quelques secondes après avoir cliqué sur les boutons avant de fermer l'onglet.

## "Accès refusé" / "Vous n'avez pas la permission"

Cela signifie généralement l'un des éléments suivants :
- votre rôle n'autorise pas cette page/action
- vous n'avez pas accès au pays/organisation pour cette mission

Que faire :
- Contactez votre administrateur et partagez :
  - la page que vous avez essayé d'ouvrir
  - le nom de la mission (si pertinent)
  - une capture d'écran du message

## Boutons manquants (Modifier / Soumettre / Approuver)

Cela est généralement causé par :
- le statut de soumission (soumis/approuvé peut être verrouillé), ou
- des permissions manquantes (rôle non assigné)

Voir : [Statuts de soumission et ce que vous pouvez faire](submission-statuses-and-permissions.md)

## Problèmes de sauvegarde (progression manquante)

Si vous pensez que les réponses n'ont pas été enregistrées :
- Recherchez un bouton "Enregistrer brouillon" (s'il est présent) et utilisez-le avant de partir.
- Évitez de fermer l'onglet juste après avoir apporté de nombreux changements.
- Si la page a gelé, actualisez une fois et vérifiez si vos réponses sont toujours là.

Lorsque vous contactez le support, partagez :
- nom de la mission
- heure approximative à laquelle vous avez saisi les données
- quelles sections ont été affectées

## Problèmes de téléchargement de fichiers (documents justificatifs)

Causes courantes :
- le fichier est trop volumineux
- le type de fichier n'est pas autorisé
- votre rôle n'autorise pas le téléchargement de documents

Voir : [Documents justificatifs (Point focal)](../focal-point/supporting-documents.md).
