# ابدأ من هنا

استخدم هذه الصفحة إذا كنت جديداً وتريد فهم ما يجب فعله أولاً.

## البدء السريع

1. افتح **لوحة التحكم** الخاصة بك.
2. حدد **اللغة** في [إعدادات الحساب](account-settings.md).
3. افتح [عرض مهامك](../focal-point/view-assignments.md) للعثور على النماذج التي تحتاج إلى إكمالها.
