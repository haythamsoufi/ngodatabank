# Obtener ayuda

Usa esta página si estás atascado y necesitas ayuda.

## Primero, prueba estas verificaciones rápidas

- **Actualiza la página** e intenta de nuevo (pueden ocurrir problemas temporales).
- Asegúrate de estar en el **país correcto** (si tu trabajo depende del acceso al país).
- Verifica tu **conexión a Internet**.

## Si no puedes ver algo que esperabas

- Si eres un punto focal y **no ves asignaciones**, es posible que aún no estés asignado a un país.
- Si puedes iniciar sesión pero **no ves los elementos de menú correctos**, tu rol/permisos pueden estar limitados.

## Dónde encontrar orientación

- **Comienza aquí**: ver [Cómo funciona la plataforma](../../getting-started/how-it-works.md)
- **Definiciones**: ver [Glosario](../../getting-started/glossary.md)
- **Guías paso a paso**: ver [Guías de usuario](../README.md)

## Cómo pedir soporte (información recomendada)

Al contactar soporte, incluye:

- Lo que estabas intentando hacer (1-2 oraciones)
- La página en la que estabas (copia la URL si es posible)
- Tu rol (administrador o punto focal)
- Tu país/países (si es relevante)
- Una captura de pantalla del mensaje/error
- La hora en que ocurrió
