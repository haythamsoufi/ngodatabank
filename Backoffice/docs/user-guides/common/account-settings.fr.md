# Paramètres du compte

Utilisez cette page pour mettre à jour votre profil et vos préférences.

## Ce que vous pouvez mettre à jour

- **Profil** : nom, titre (titre du poste/rôle)
- **Couleur du profil** : choisissez une couleur pour l'icône de votre profil
- **Assistant de chat IA** : activer ou désactiver l'assistant de chat IA (si disponible)
- **Appareils enregistrés** : voir et gérer vos appareils mobiles enregistrés pour les notifications push

## Ce que vous ne pouvez pas changer ici

- **Adresse e-mail** : Votre e-mail est géré par SSO et ne peut pas être modifié. Contactez votre administrateur système.
- **Mot de passe** : Les changements de mot de passe doivent être effectués par un administrateur système. Contactez votre administrateur pour changer votre mot de passe.
- **Rôles** : Vos rôles sont gérés par les administrateurs système.

## Mettre à jour votre profil

1. Ouvrez **Paramètres du compte** (depuis votre tableau de bord ou le menu utilisateur).
2. Mettez à jour votre **Nom** et **Titre** (optionnel).
3. Choisissez une **Couleur de profil** pour l'icône de votre profil.
4. Si disponible, activez ou désactivez **Assistant de chat IA**.
5. Cliquez sur **Enregistrer les modifications**.

## Gérer les appareils enregistrés

Vous pouvez voir et gérer vos appareils mobiles enregistrés :

- **Voir les appareils** : Voir tous les appareils enregistrés pour les notifications push
- **Terminer la session** : Se déconnecter d'un appareil spécifique (garde l'appareil enregistré)
- **Supprimer l'appareil** : Supprimer définitivement un appareil du registre

**Note** : Pour changer votre adresse e-mail ou votre mot de passe, veuillez contacter votre administrateur système.

## Conseils

- Si vous changez votre langue, certaines pages peuvent encore apparaître en anglais jusqu'à ce qu'elles soient traduites.
- Gardez vos informations de contact à jour.
