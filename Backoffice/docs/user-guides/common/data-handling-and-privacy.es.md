# Manejo de datos y privacidad (orientación básica)

Usa esta página para reducir el riesgo al recopilar, exportar y compartir datos.

## Qué evitar en los envíos

A menos que sea explícitamente solicitado y aprobado por tu proceso:
- evita identificadores personales (nombres, números de teléfono, direcciones)
- evita adjuntar documentos con datos personales sensibles

Si se requieren datos sensibles, acuerda:
- quién puede acceder a ellos
- cuánto tiempo se conservan
- cómo se comparten de forma segura

## Documentos de respaldo (archivos de evidencia)

Buenas prácticas:
- usa nombres de archivo claros (ejemplo: `Country_Period_Source.pdf`)
- sube el mínimo necesario
- elimina identificadores personales de los documentos cuando sea posible

Ver: [Documentos de respaldo (Punto focal)](../focal-point/supporting-documents.md).

## Archivos de exportación (Excel/CSV)

Trata las exportaciones como sensibles:
- no compartas mediante enlaces públicos
- almacénalas en ubicaciones aprobadas (unidades compartidas con control de acceso)
- mantén una copia "exportación sin procesar" sin cambios
- documenta cualquier paso de limpieza manual

## Envíos de URL públicas (riesgo especial)

Las URL públicas pueden compartirse ampliamente.

Si las usas:
- comparte solo a través de canales de confianza
- monitorea los envíos regularmente
- desactiva el enlace cuando termine el período de recopilación

Si tu proceso usa enlaces públicos, sigue la orientación de tu organización. Pregunta a tu administrador cómo se configuran los envíos por URL pública y cuándo deben desactivarse los enlaces.
