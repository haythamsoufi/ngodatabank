# Conceptos básicos de solución de problemas (todos)

Usa esta página para correcciones rápidas antes de escalar a un equipo de administración/soporte.

## Primeras verificaciones (rápidas)

- Actualiza la página.
- Cierra sesión e inicia sesión de nuevo.
- Prueba otro navegador (si es posible).
- Si estás en una conexión lenta, espera unos segundos después de hacer clic en los botones antes de cerrar la pestaña.

## "Acceso denegado" / "No tienes permiso"

Esto generalmente significa uno de:
- tu rol no permite esa página/acción
- no tienes acceso al país/organización para esa asignación

Qué hacer:
- Contacta a tu administrador y comparte:
  - la página que intentaste abrir
  - el nombre de la asignación (si es relevante)
  - una captura de pantalla del mensaje

## Botones faltantes (Editar / Enviar / Aprobar)

Esto generalmente es causado por:
- el estado del envío (enviado/aprobado puede estar bloqueado), o
- permisos faltantes (rol no asignado)

Ver: [Estados de envío y qué puedes hacer](submission-statuses-and-permissions.md)

## Problemas de guardado (progreso faltante)

Si crees que las respuestas no se guardaron:
- Busca un botón "Guardar borrador" (si está presente) y úsalo antes de salir.
- Evita cerrar la pestaña justo después de hacer muchos cambios.
- Si la página se congeló, actualiza una vez y verifica si tus respuestas siguen ahí.

Al contactar soporte, comparte:
- nombre de la asignación
- hora aproximada en que ingresaste los datos
- qué secciones se vieron afectadas

## Problemas de carga de archivos (documentos de respaldo)

Causas comunes:
- el archivo es demasiado grande
- el tipo de archivo no está permitido
- tu rol no permite la carga de documentos

Ver: [Documentos de respaldo (Punto focal)](../focal-point/supporting-documents.md).
