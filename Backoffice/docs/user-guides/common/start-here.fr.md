# Commencez ici

Utilisez cette page si vous êtes nouveau et que vous voulez comprendre ce qu'il faut faire en premier.

## Démarrage rapide

1. Ouvrez votre **Tableau de bord**.
2. Définissez votre **Langue** dans [Paramètres du compte](account-settings.md).
3. Ouvrez [Voir vos missions](../focal-point/view-assignments.md) pour trouver les formulaires à compléter.
