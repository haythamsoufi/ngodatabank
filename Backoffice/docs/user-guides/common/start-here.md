# Start here

Use this page if you’re new and want to understand what to do first.

## Quick start

1. Open your **Dashboard**.
2. Set your **Language** in [Account settings](account-settings.md).
3. Open [View your assignments](../focal-point/view-assignments.md) to find forms you need to complete.
