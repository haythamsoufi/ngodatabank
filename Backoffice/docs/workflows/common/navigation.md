# Platform Navigation

This guide helps you navigate the main sections and features of the platform.

## Prerequisites

- Logged in to the system

## Main Navigation Areas

### For All Users

#### Dashboard
- **Page**: `/`
- **Selector**: `a[href="/"], .sidebar-item-container[href="/"]`
- **Help**: Your personalized home page showing pending assignments, recent activity, and quick access to common tasks.

#### Account Settings
- **Page**: `/account-settings`
- **Selector**: `a[href*="account-settings"], a[href*="account_settings"], .profile-dropdown`
- **Help**: Update your profile, change password, and set notification preferences.

### For Focal Points

#### My Assignments
- **Page**: `/`
- **Selector**: `.bg-white.p-6.rounded-lg.shadow-md, .grid.gap-4`
- **Help**: View all your pending and completed form submissions on your dashboard.

#### Data Entry
- **Page**: `/forms/assignment`
- **Selector**: `a[href*="/forms/assignment/"]`
- **Help**: Click on an assignment to access the data entry form for your assigned countries.

### For Administrators

#### User & Access Management
- **Page**: `/admin/users`
- **Selector**: `a[href="/admin/users"], .nav-users`
- **Help**: Manage user accounts, roles, and permissions. Add new users or modify existing ones.

#### Form & Data Management
- **Page**: `/admin/templates`
- **Selector**: `a[href="/admin/templates"], .nav-templates`
- **Help**: Create and manage form templates. Design forms with sections and fields.

#### Manage Assignments
- **Page**: `/admin/assignments`
- **Selector**: `a[href="/admin/assignments"], .nav-assignments`
- **Help**: Create and track form assignments. Monitor submission progress across countries.

#### Reference Data
- **Page**: `/admin/indicators`
- **Selector**: `a[href="/admin/indicators"], .nav-indicators`
- **Help**: Manage the Indicator Bank with standardized data definitions and metrics.

#### Countries
- **Page**: `/admin/countries`
- **Selector**: `a[href="/admin/countries"], .nav-countries`
- **Help**: Configure countries, regions, and National Society information.

#### Analytics & Monitoring
- **Page**: `/admin/analytics`
- **Selector**: `a[href="/admin/analytics"], .nav-analytics`
- **Help**: View dashboards, reports, and data visualizations. Monitor platform usage and data quality.

## Quick Navigation Tips

### Using the Sidebar
The main sidebar provides access to all major sections. Click on a category to expand it and see sub-items.

### Using Search
- Press `/` or click the search icon to open quick search
- Type to find pages, users, templates, or countries
- Press Enter to navigate to the first result

### Keyboard Shortcuts
| Shortcut | Action |
|----------|--------|
| `?` | Show keyboard shortcuts |
| `/` | Open search |
| `g d` | Go to Dashboard |
| `g s` | Go to Settings |

### Breadcrumbs
Use the breadcrumb trail at the top of pages to navigate back to parent sections.

## Tips

- Bookmark frequently used pages for quick access
- Use the sidebar collapse button for more screen space
- Check the notification bell for updates and reminders
- The chatbot can help you find any feature - just ask!

## Getting Help

If you can't find something:
1. Use the search feature
2. Ask the chatbot "Where is [feature]?"
3. Check the help documentation
4. Contact your administrator
