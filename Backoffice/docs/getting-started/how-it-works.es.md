# Cómo funciona la plataforma

Esta página explica las piezas principales del Backoffice en términos simples.

## Ideas clave (lenguaje simple)

- **Plantilla**: la estructura del formulario (preguntas + secciones) que crean los administradores.
- **Asignación**: cuando un administrador envía una plantilla a uno o más países para completar.
- **Envío de formulario**: cuando un punto focal termina el formulario y hace clic en **Enviar**.

## Flujo de trabajo típico

1. Un **administrador** crea o actualiza una **plantilla**.
2. El administrador crea una **asignación** usando esa plantilla y selecciona el país o países.
3. Un **punto focal** abre la asignación, completa el formulario, corrige los mensajes de validación y hace clic en **Enviar**.
4. Los administradores pueden monitorear el progreso (por ejemplo: pendiente / en progreso / enviado).

## Dónde ir a continuación

- Si eres un punto focal: comienza con **Guías de usuario → Punto focal → Ver tus asignaciones**.
- Si eres administrador: comienza con **Guías de usuario → Administrador → Crear y gestionar asignaciones**.
