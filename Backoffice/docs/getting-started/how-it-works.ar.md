# كيف يعمل النظام

تشرح هذه الصفحة الأجزاء الرئيسية للمكتب الخلفي بعبارات بسيطة.

## الأفكار الرئيسية (بلغة بسيطة)

- **القالب**: هيكل النموذج (الأسئلة + الأقسام) الذي ينشئه المسؤولون.
- **المهمة**: عندما يرسل المسؤول قالباً إلى دولة واحدة أو أكثر للإكمال.
- **إرسال النموذج**: عندما ينتهي نقطة الاتصال من ملء النموذج وينقر على **إرسال**.

## سير العمل النموذجي

1. ينشئ **المسؤول** أو يحدّث **قالباً**.
2. ينشئ المسؤول **مهمة** باستخدام ذلك القالب ويختار الدولة/الدول.
3. تفتح **نقطة الاتصال** المهمة وتملأ النموذج وتصلح رسائل التحقق وتنقر على **إرسال**.
4. يمكن للمسؤولين مراقبة التقدم (على سبيل المثال: معلق / قيد التنفيذ / تم الإرسال).

## أين تذهب بعد ذلك

- إذا كنت نقطة اتصال: ابدأ بـ **دليل المستخدم → نقطة الاتصال → عرض مهامك**.
- إذا كنت مسؤولاً: ابدأ بـ **دليل المستخدم → المسؤول → إنشاء وإدارة المهام**.
