# Getting Started

This section helps you get oriented and start using the Backoffice.

## Quick start

- For step-by-step instructions, start with [Start here](../user-guides/common/start-here.md).
- If you’re new, read [How the platform works](how-it-works.md) first.

## What you can do here

- Understand the main areas of the platform
- Learn the core workflows for your role
- Find where things are in the UI

## First steps

1. Sign in and open your **Dashboard**
2. Set your **Language** in [Account settings](../user-guides/common/account-settings.md)
3. Open [View your assignments](../user-guides/focal-point/view-assignments.md) to find forms you need to complete.

## Need help?

- See [Start here](../user-guides/common/start-here.md) and the guides in the documentation sidebar
- See [Getting help](../user-guides/common/getting-help.md)
- If you still need help, contact your administrator or support channel for your organization
