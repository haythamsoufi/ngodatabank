# Para comenzar

Esta sección te ayuda a orientarte y comenzar a usar el Backoffice.

## Inicio rápido

- Para instrucciones paso a paso, comienza con [Comienza aquí](../user-guides/common/start-here.md).
- Si eres nuevo, lee primero [Cómo funciona la plataforma](how-it-works.md).

## Qué puedes hacer aquí

- Entender las áreas principales de la plataforma
- Aprender los flujos de trabajo principales para tu rol
- Encontrar dónde están las cosas en la interfaz

## Primeros pasos

1. Inicia sesión y abre tu **Panel de control**
2. Establece tu **Idioma** en [Configuración de cuenta](../user-guides/common/account-settings.md)
3. Abre [Ver tus asignaciones](../user-guides/focal-point/view-assignments.md) para encontrar los formularios que debes completar.

## ¿Necesitas ayuda?

- Consulta [Comienza aquí](../user-guides/common/start-here.md) y las guías en la barra lateral de documentación
- Ver [Obtener ayuda](../user-guides/common/getting-help.md)
- Si aún necesitas ayuda, contacta a tu administrador o canal de soporte de tu organización
